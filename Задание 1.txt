Из методов list 5-ка самых популярных:
1. list1.append(sub) - Добавляет элемент 'sub' в конец списка 'list1', если передаем другой список, то он добавляется как вложенный список.
2. list2.extend(sub) - Добавляет все элементы списка 'sub' в другой список 'list2'.
3. count(sub) - Считает сколько раз значение 'sub' появляется в списке
4. pop(2) - Удаляет и возвращает значение по указанному индексу '2'.
5. index(sub) - Находит индекст первого соответсвующего элемента 'sub'.

Из методов set 5-ка самых популярных:
1.set1.add(sub) - Добавляет один элемен 'sub' в 'set1'.
2.set2.update(sub) - Lобавляет несколько элементов 'sub' в 'set2'.
3.set3.discard(sub) - Удаляет конкретный элемент и не возвращает ошибку, если тот не был найден во множестве 'set3'.
4.set4.remove(sub) - Удаляет из 'set4' конкретный элемент 'sub' и возвращает ошибку в том случае, если его нет в 'set4'.
5.set5.pop() - Удаляет по одному элементу за раз в случайном порядке. Метод pop()не требует аргументов/

Из методов dict 5-ка самых популярных:
1.dict1.update() - Обновляет словарь, еслт в него нужно добавить новый ключ и знаечение.
2.dict2.clear() - Очищает словарь
3.dict3.copy() - Возвращает копию словаря
4.dict4.pop(key, default]) - Удаляет ключ и возвращает значение. Если ключа нет, возвращает default
5.dict5.popitem() - Удаляет и возвращает пару (ключ, значение) с конца словаря. Если словарь пуст, бросает исключение KeyError.


Из методов str 5-ка самых популярных:
1. str1.lower() - переводит строку 'str1' в нижний регистр.
2. str2.count(sub, star, end) - Показывает количество подстроки 'sub' в строке 'str2'. 'start' и 'end' это диапозон поиска, параметры не обязательные.
3. str3.replace(old, new, count=-1) - Заменяет подстроку 'old' в строке 'str3' на 'new'. Параметр 'count=-1' это количество замен, если установит параметр 2, поменяются первые 2 подстроки.
4. str4.split(sep, maxsplit) - Разбивает строку 'str4' на список строк и разделяет его. По умолчанию разделитель пробел, но в 'sep' можно указать любой, второрой параметр 'maxsplit' устанавливает количество указанных разделителей он не обязатаельный как и парамерт 'sep'.
5. str5.join(iterable) - Объединяет строки 'str5' и 'iterable', причем строка 'a' будет разделителем строки 'iterable'.
